📑 Composant Tabs

Navigation par onglets interactive et élégante.

Les tabs permettent une navigation fluide entre plusieurs sections de contenu. Cliquez sur les onglets pour voir la magie!

  • ✨ Animation fluide
  • 🎨 Design moderne
  • 📱 Responsive
  • ♿ Accessible

Consultez la documentation complète sur GitHub pour plus d'informations.

<div class="kt-tabs">
  <ul class="kt-tabs-nav">
    <li><button class="kt-tab-link kt-active">Tab 1</button></li>
  </ul>
</div>

📂 Composant Accordion

Contenu dépliable pour une meilleure organisation de l'information.

KitokoCSS est un framework CSS moderne et esthétique conçu pour accélérer votre développement front-end.

Avec KitokoCSS, vous bénéficiez de composants prêts à l'emploi, d'une grande flexibilité et d'une performance optimale.

Incluez simplement le CDN dans votre HTML et commencez à utiliser les classes kt-!

<div class="kt-accordion">
  <div class="kt-accordion-item">
    <button class="kt-accordion-header">Title</button>
  </div>
</div>
